QuickCart Post‑Deploy Full QA Checklist

Confirm the site is safe for real traffic and sales.

00. Deploy Finish

composer install --no-dev --optimize-autoloader
npm ci && npm run build
php artisan migrate --force
php artisan quickcart:ensure-storage
php artisan cache:clear
php artisan config:cache
php artisan route:cache
php artisan view:cache
Verify APP_ENV, APP_DEBUG=false, APP_URL, storage permissions and public/storage symlink.

01. Health & Assets

  • GET /up = 200
  • No blank screen
  • No React/Inertia errors
  • Assets load from /build/assets
  • Images return 200
